Message Metadata

A Message can have a set of metadata associated with it.

Retrieving metadata

You can retrieve the metadata from a message using either:

  • getMetadata(Class<?> c) : get a specific type of metadata

  • getMetadata() : allows iterate through the metadata (it also provides method to manipulate the set)

// retrieve some metadata if present
Optional<MyMetadata> myMetadata = message.getMetadata(MyMetadata.class);
message.getMetadata().forEach(metadata -> {
    // iterate through the set of metadata
});

Adding metadata

Metadata associated with a message can be any object. Note that only one instance of each class can be attached. So, it is recommend to create specific class to store your metadata. Also, because the metadata may be accessed concurrently, it is highly recommended to use immutable objects.

You can create a message with some metadata using Message.of(payload, Metadata.of(…​) )

MyMetadata metadata = new MyMetadata("author", "me");
Message<String> msg = Message.of("hello", Metadata.of(metadata));

You can also add metadata to an existing message using the withMetadata method.

Metadata propagation

When your are processing payloads, Reactive Messaging propagates the metadata through the stream. So, it copies the metadata from the incoming message to the outgoing message.

@Outgoing("source")
public Multi<Message<Integer>> generate() {
    return Multi.createFrom().range(0, 10)
        .map(i -> Message.of(i, Metadata.of(new MyMetadata("author", "me"))));
}

@Incoming("source")
@Outgoing("sink")
public int increment(int i) {
    return i + 1;
}

@Outgoing("sink")
public CompletionStage<Void> consume(Message<Integer> in) {
    Optional<MyMetadata> metadata = in.getMetadata(MyMetadata.class);
    return in.ack();
}

When dealing with Message or streams, the propagation is manual:

@Outgoing("source")
public Multi<Message<Integer>> generate() {
    return Multi.createFrom().range(0, 10)
        .map(i -> Message.of(i, Metadata.of(new MyMetadata("author", "me"))));
}

@Incoming("source")
@Outgoing("sink")
public Message<Integer> increment(Message<Integer> in) {
    return in.withPayload(in.getPayload() + 1);
}

@Outgoing("sink")
public CompletionStage<Void> consume(Message<Integer> in) {
    Optional<MyMetadata> metadata = in.getMetadata(MyMetadata.class);
    return in.ack();
}

Consult the signature list to check which signature support automatic propagation.
